TrexeMe

Συχνές Ερωτήσεις

Τι είναι το TrexeMe?

Το TrexeMe είναι μια σελίδα όπου μπορείς να τρέχεις προγράμματα γραμμένα στη ΓΛΩΣΣΑ. Εδώ οι μαθητές της Γ’ (και της Β’) Λυκείου μπορούν να εξασκηθούν πρακτικά, να δοκιμάσουν τις διάφορες ιδέες τους, ώστε να αποκτήσουν εμπειρία και βαθύτερη κατανόηση στον προγραμματισμό.

Τι είναι η "ΓΛΩΣΣA"?

Η "ΓΛΩΣΣΑ" είναι μια γλώσσα προγραμματισμού που αναπτύχθηκε με σκοπό την εισαγωγή των μαθητών του Λυκείου στον προγραμματισμό.

Πως χρησιμοποιώ τη σελίδα?

Αρχικά γράφεις τον κώδικα σου στο αριστερό πλαίσιο στην αρχική σελίδα. Για να τρέξεις το πρόγραμμα σου πατάς το κουμπί RUN που βρίσκεται πάνω πάνω στη σελίδα. Το δεξιό πλαίσιο είναι η οθόνη όπου γράφεται η έξοδος του προγράμματος, τυχόν σφάλματα και από εκεί μπορείς να εισάγεις πληροφορίες από το πληκτρολόγιο.

Το κουμπί Save αποθηκεύει τον κώδικα σου τοπικά στον υπολογιστή σου σε ένα αρχείο τύπου txt.

To κουμπί Load φορτώνει αρχεία κειμένου από τον υπολογιστή σου στη σελίδα.

Τέλος, το κουμπί Code ανοίγει μια λίστα από έτοιμα παραδείγματα κώδικα από τα οποία μπορείς να επιλέξεις.
